关于

出怪列表编辑器（Zombie List Editor），用于显示和编辑出怪列表。
使用本工具需要Microsoft Office Excel 2007及更高版本。打开时如果有提示安全警告，选择“启用宏”。

“读取列表”按钮用于将游戏内存中的出怪列表写入Excel中。Excel的中间区域是可编辑的，可通过手动在中间区域输入数字来逐波编辑出怪列表，点击“写入列表”按钮以将表格写入游戏内存。
默认按照僵尸种类从小到大写入游戏内存，此顺序会影响僵尸的栈位和选行。如需要精确控制各僵尸在列表中的顺序，请使用“逐波编辑”功能。
